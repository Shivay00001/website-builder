Automatic Website Builder

A Python Tkinter application that generates professional, responsive static websites from user input. It supports multiple templates including Business, Portfolio, E-commerce, Blog, and Landing Page.

Features

  • GUI Interface: Easy-to-use form for entering website details.
  • Multiple Templates:
    • Business (Corporate style)
    • Portfolio (For creatives)
    • E-commerce (Product showcase)
    • Blog (Article layout)
    • Landing Page (Conversion focused)
  • Live Preview: See the generated HTML code instantly.
  • Browser Preview: Open the generated site in your default browser.
  • Customization: Choose primary colors and input social media links.
  • Export: Save the generated website as a standard HTML file.

Usage

python website_builder.py

Dependencies

  • tkinter (Standard Python library)
